## 介紹：針對深度學習推薦模型在大量稀疏特徵資料無法完全放入記憶體時，必須頻繁存取外部儲存的瓶頸問題，利用OCSSD的高平行性與可控資料放置特性進行最佳化，降低記憶體與儲存間的資料搬移與寫入放大。
### 主要貢獻：
- 將原本僅支援 4 PU(平行單位) 的環境擴充，觀察不同平行度下對深度學習推薦模型訓練時間與執行效率的影響。
- 針對 Embedding Cache Management，建立統計機制以分別計算讀取與寫入的快取命中率 (cache hit rate)，進一步分析 I/O 。
- 計算不同模型中各個 chunk 的 Invalid Ratio (單位為page)以及總寫入量，評估寫入放大與碎片化情形。
